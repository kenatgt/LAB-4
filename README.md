# LAB-4
Laboratorio 4 Polimorfismo: CMS

## Ejecución rápida
1. Requisitos: JDK 17+ en el `PATH`.
2. Compilar todos los archivos:
   ```bash
   javac $(find src -name "*.java")
   ```
3. Ejecutar el driver principal:
   ```bash
   java -cp src view.MainCMS
   ```

El flujo imprime en consola el resultado de las operaciones principales del CMS (altas, filtros, visualizaciones y reporte final).
